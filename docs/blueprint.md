# **App Name**: Peak Progress

## Core Features:

- Daily Climb Log: A form for each family member to log whether they climbed the mountain each day.
- Location-Based Status: Display a happy or sad symbol on a map for each day, based on whether the climb was logged.
- Progress Visualization: Visualize each family member's progress over time, showing trends and streaks.

## Style Guidelines:

- Primary color: Deep green (#386641) to represent nature and growth.
- Background color: Light beige (#F5F5DC) to provide a neutral backdrop.
- Accent color: Burnt orange (#A8511C) to highlight important information and actions.
- Body and headline font: 'PT Sans', a humanist sans-serif to combine a modern look with a little warmth and readability.
- Use simple, clear icons to represent climb status (e.g., a mountain silhouette for 'climbed,' a flat line for 'not climbed').
- Display a calendar-based view for easy navigation through daily logs.
- Use subtle animations to transition between days and display status changes.
